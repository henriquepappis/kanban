# Kanban

Sistema 2 - Sistema de Atividades Internas
O analista de sistemas pediu para criar um sistema para gerenciar as atividades internas de qualquer equipe da empresa.
- O sistema deve ter um cadastro de usuários simples (login e senha);
- O sistema deve ter opções para visualizar, cadastrar, editar e excluir as atividades;
- A atividade deve ter um nome;
- A atividade deve ter um status (To Do, Doing, Done);
- Nenhum usuário pode ter 2 atividades "Doing" ao mesmo tempo;
